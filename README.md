# PeylixNvim

My Neovim config based on AstroNvim v5.

## Dependencies

1. Git
2. ripgrep
3. Node.js & npm
4. Cargo
5. Tree-sitter CLI
6. Nerd Font (for icons)
7. macism (for `im-select.nvim` on macOS)
8. Python 3.10+
   - `pynvim` package
   - `jupyter_client` package

## Reduced Profile

To start nvim with reduced config, run:

```bash
export PNVIM_PROFILE=reduced
```

Or add it to the shell config file (e.g., `~/.bashrc`, `~/.zshrc`)
